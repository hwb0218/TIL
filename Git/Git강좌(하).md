## Repository 생성
- github에서 Repository생성 
- git status로 현재까지의 사항들이 모두 commit되었는지 확인
- git remote - 현 폴더의 원격 repository를 확인
- git remote add origin(변경가능) (github주소) - 해당 repository를 현재 폴더의 origin이라는 이름의 원격 저장소로 설정한다 
- git push -u origin main - 폴더의 현 브랜치에 커밋된 내용을 origin이라는 repository의 main이란 브랜치에 올리겠다
